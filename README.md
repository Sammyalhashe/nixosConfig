# NixOS Configuration

## Mango Session Keybindings

| Keybinding | Action |
| :--- | :--- |
| **General** | |
| `Super` + `Space` | Open App Launcher (Wofi) |
| `Super` + `T` / `Return` | Open Terminal (Alacritty) |
| `Super` + `W` / `Backspace` | Close Focused Window |
| `Super` + `V` | Toggle Floating Mode |
| `Super` + `Shift` + `Return` | Toggle Fullscreen |
| **Navigation** | |
| `Super` + `Arrows` / `h/j/k/l` | Move Focus |
| `Super` + `1-0` | Switch Workspace |
| `Super` + `Shift` + `1-0` | Move Window to Workspace |
| **Session** | |
| `Super` + `Escape` | Lock Screen |
| `Super` + `Shift` + `Escape` | Quit Session (Logout) |
| `Super` + `Ctrl` + `Escape` | Reboot |
| `Super` + `Shift` + `Ctrl` + `Escape` | Poweroff |
| **Media & Hardware** | |
| `Print` | Screenshot (Region) |
| `Shift` + `Print` | Screenshot (Window) |
| `Ctrl` + `Print` | Screenshot (Output) |
| `Super` + `Print` | Color Picker |
| `Media Keys` | Volume / Brightness / Playback Control |
